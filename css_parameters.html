CSS参数可视化

background

background 属性用于设置元素的背景，包括颜色、图片、渐变等。
这是一个使用 linear-gradient 的背景效果
.background-demo { background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); color: white; padding: 40px; text-align: center; border-radius: 12px; }

backdrop-filter

backdrop-filter 属性用于为元素后面的内容添加模糊效果，常用于创建毛玻璃效果。
这是一个使用 backdrop-filter: blur(10px) 的毛玻璃效果
.backdrop-filter-content { background: rgba(255, 255, 255, 0.7); backdrop-filter: blur(10px); -webkit-backdrop-filter: blur(10px); padding: 20px; border-radius: 8px; text-align: center; }

border-radius

border-radius 属性用于设置元素的圆角，值越大，圆角效果越明显。
这是一个使用 border-radius: 20px 的圆角效果
.border-radius-demo { background: #667eea; color: white; padding: 40px; text-align: center; border-radius: 20px; }

padding

padding 属性用于设置元素的内边距，控制元素内容与边框之间的距离。
这是一个使用 padding: 30px 的内边距效果
.padding-demo { background: #667eea; color: white; padding: 30px; text-align: center; border-radius: 8px; }

margin-bottom

margin-bottom 属性用于设置元素的下外边距，控制元素与下方元素之间的距离。
这是一个使用 margin-bottom: 40px 的下外边距效果
下方元素
.margin-bottom-demo { background: #667eea; color: white; padding: 20px; text-align: center; border-radius: 8px; margin-bottom: 40px; }

box-shadow

box-shadow 属性用于为元素添加阴影效果，增强元素的立体感。
这是一个使用 box-shadow: 0 10px 20px rgba(0, 0, 0, 0.2) 的阴影效果
.box-shadow-demo { background: white; color: #2c3e50; padding: 40px; text-align: center; border-radius: 8px; box-shadow: 0 10px 20px rgba(0, 0, 0, 0.2); }

border

border 属性用于设置元素的边框，包括宽度、样式和颜色。
这是一个使用 border: 4px solid #667eea 的边框效果
.border-demo { background: white; color: #2c3e50; padding: 40px; text-align: center; border: 4px solid #667eea; border-radius: 8px; }

transition

transition 属性用于设置元素的过渡效果，使元素的变化更加平滑。
鼠标悬停时会有平滑的过渡效果
.transition-demo { background: #667eea; color: white; padding: 40px; text-align: center; border-radius: 8px; transition: all 0.3s ease; cursor: pointer; } .transition-demo:hover { background: #764ba2; transform: scale(1.05); }
